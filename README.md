# Asynchronous JavaScript & RESTful APIs

## Weather Dashboard

A real-time Weather Dashboard built using JavaScript Fetch API and OpenWeather REST API.

---

## Features

- Search weather by city
- Real-time API requests
- Async/Await
- Fetch API
- Error Handling
- JSON Parsing
- Dynamic DOM Rendering

---

## Technologies Used

- HTML5
- CSS3
- JavaScript ES6
- REST API
- Fetch API

---

## Installation

1. Clone repository

```bash
git clone https://github.com/YOUR_USERNAME/asynchronous-javascript-weather-dashboard.git
```

2. Open project

```bash
cd asynchronous-javascript-weather-dashboard
```

3. Add your API key in script.js

```javascript
const API_KEY = "YOUR_API_KEY";
```

4. Run

Open index.html in browser.
